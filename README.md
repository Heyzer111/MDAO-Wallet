# MDAO-Wallet

## Hướng dẫn sử dụng

### 1. Tạo file `1.txt`
- Dán `queryid` vào mỗi dòng, mỗi dòng tương ứng với một tài khoản.

### 2. Cài đặt các thư viện cần thiết
- Chạy lệnh sau để cài đặt các thư viện:
    ```bash
    pip install requests termcolor
    ```

### 3. Chạy script
- Để chạy script, sử dụng lệnh sau:
    ```bash
    python mdao.py
    ```
